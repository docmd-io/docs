---

## 问题
平坦或者无组织的 Markdown 文件夹导致路径混乱与 URL 管理的失控。

## 为什么重要
不清晰的组织架构不仅妨碍路由的美观，更导致作者难以找到应该更修的原始文件位置。

## 方法
采用 Diátaxis 方法论。将概念按照“教程”、“指南”、“参考”、“架构详解”分隔，与操作系统的目录一一对应。

## 实施
设定精确的多级分层。
结合各业务模块的 `navigation.json` 管理他们的自主页面展示，而非通过全局。

## 权衡
高级的内容分类法提高了文档贡献门槛。作者经常困惑于“当前内容属于进阶指南还是初学者教程？”。
