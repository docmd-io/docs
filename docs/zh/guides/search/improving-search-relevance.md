title: "通过规范化标注获得出色的检索相关性"
description: "用结构优化权重与相关性指南。"
---

## Problem
机器不会自己看懂文章质量。不含有核心短语的大纲将会获得糟糕低分的权重分数。

## 为什么重要
精确把控词语击中区域能够显著将您的精华内容往首页提升而非沉没第三页海底。

## Ansatz
按照搜索模型偏好规则强化标题和子头文字，主动放弃普通叙事。

## Implementation
在 Frontmatter 内使用 keywords 隐式注入同义词根，同时给与加粗代码 `<strong>` 高光凸显给文本标注优先权重标记。

## Trade-offs
过度运营元数据将非常内耗管理时间。改变布局有时破坏干净的排版。
