## 问题
全量的几十兆编译检索 JSON 卡住加载网页导致浏览器主线程报错而崩溃卡顿无响应。

## 为什么重要
快捷地通过命令行风格 Cmd+K 获取指令解答是效率工具存留用户不离开的心智。

## 方法
分发索引，懒惰加载，并且舍弃边缘页等策略进行 JSON 精简减负。

## 实施
剔除介词，停止代码组件内纯符号的标记污染，只使用按版本并针对国家本地化生成的部分缓存搜索字典文件。

## 权衡
有很少用的页面无法直接借此跳板送达。由于网络慢也会造成第一下检索时微微的白光加载错觉闪屏。
