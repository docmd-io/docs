# docmd

一条命令将 Markdown 转化为生产级文档。静态 HTML 助力 SEO。SPA 提升速度。默认支持 AI。

::: button "快速开始" ./getting-started/quick-start.md icon:rocket
::: button "GitHub" external:https://github.com/docmd-io/docmd color:#24292e icon:github
:::

## 概览

docmd 是一个零配置的文档生成器。它能从 Markdown 文件直接生成高性能的静态网站。

```bash
npx @docmd/core dev
```

只需一条命令。引擎自动构建网站、生成导航并启用即时搜索。
